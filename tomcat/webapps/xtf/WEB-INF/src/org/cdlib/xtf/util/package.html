The 'util' package contains a number of general utility classes used by other code throughout XTF. They can be divided into the following categories:

  1. Data Structures / Iterators
    • ArrayIterator
    • AttribList
    • Attrib
    • CircularQueue
    • ConsecutiveMap
    • EmbeddedList
    • Linkable
    • LinkableImpl
  2. Encoding/Decoding
    • Base64
  3. Fast Tokenization
    • FastTokenizer
    • FastStringReader
    • CheckingTokenStream
  4. Structured Disk Files
    • StructuredFile
    • Subfile
    • PackedByteBuf
    • DiskHashReader
    • DiskHashWriter
  5. XML-Related
    • XMLWriter
    • XTFSaxonErrorListener
  6. Input/Output and Files
    • LimitedOutputStream
    • Path
  7. Other
    • GeneralException
    • Prime
    • Tester
    • Trace
