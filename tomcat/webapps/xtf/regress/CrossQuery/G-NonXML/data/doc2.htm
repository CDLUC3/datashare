Melvyl Search Tip:  Don't like using your mouse in searching? Try Command Search.
  1. Begin at the Command screen.
  2. Type in your search using index codes and search terms, using connectors (and, or, not).  For example:
    • wsu = shelley mary and shelley percy
    • wti = sound fury or light august
    • wcf = ieee and wyr = 1966
